Etiquetas de HTML

Clasificación de las etiquetas según el estándar HTML5.2

El estándar diferencia inicialmente tres tipos de etiquetas: Section, grouping content, text-level

1. Sections

Elementos de seccionado, que son los que se reflejan en un esquema de la página

Inicialmente estos elementos incluían body y los encabezados h1...h6

En html se añaden los nuevos elementos semánticos: section, header, article, footer, nav y aside

1.1. Section

Define una sección

Section es un elemento de seccionamiento genérico, diseñado para contener una parte de un documento temáticamente definida. Los contenidos de un elemento section son usualmente precedidos por un encabezado. El elemento section está pensado para agrupar temáticamente un conjunto de elementos. Si existiese la necesidad de definir un bloque con propósitos exclusimavemnte estilísticos o de programación, el elemento div debería utilizarse en su lugar.

1.2. Header

El encabezado de una sección

El elemento header contiene al encabezado de una sección o documento, donde se colocan habitualmente los encabezados, los vínculos de navegación, los formularios de búsqueda, los logos, las tablas de contenidos, las introducciones, etc. En muchos casos, los contenidos del encabezado de un documento son consistentes a lo largo de todo el sitio.

Htmlquick

1.3. Footer

El pie de una sección

El elemento footer representa al pie de una sección o documento, donde los autores habitualmente colocan firmas, información acerca del autor, información de licencias, documentos relacionados, etc. En muchos casos, los contenidos del pie de un documento son consistentes a lo largo de todo el sitio.

Htmlquick

1.4. Nav

Sección de navegación

El elemento nav es un contenedor para enlaces de navegación, que se supone proveen formas de acceder a las secciones del sitio web, así como a documentos externos. Se supone que los vínculos dentro de este elemento corresponden a un bloque de navegación principal únicamente.

Htmlquick

1.5. Article

El Elemento article de HTML (<article>) representa una composición auto-contenida en un documento, página, una aplicación o en el sitio, que se destina a distribuir de forma independiente o reutilizable, por ejemplo, en la sindicación. Podría ser un mensaje en un foro, un artículo de una revista o un periódico, una entrada de blog, un comentario de un usuario, un widget interactivo o gadget, o cualquier otro elemento independiente del contenido

1.6. Aside

El Elemento HTML Aside (<aside>) representa una sección de una página que consiste en contenido que está tangencialmente relacionado con el contenido que le rodea, que podría ser considerado independiente de ese contenido. Estas secciones son a menudo representadas como barras laterales o como inserciones y contienen una explicación al margen como una definición de glosario, elementos relacionados indirectamente, como publicidad, la biografía del autor, o en aplicaciones web, la información de perfil o enlaces a blogs relacionados. 2.1 Párrafo

Se pueden crear párrafos con las etiquetas, p, pre, blockquote

2.2 Listas

Las listas las podemos diferenciar en tres tipos, lista ordenada (<ol>), lista desordenada (<ul>), lista detalle (<dl>).
